# Decision Tree Comparison: CART vs ID3

This project implements and compares CART (Gini) and ID3 (Entropy) decision trees
using a fully automated machine learning pipeline in Google Colab.

## Contents
- Dataset: dataset.csv
- Notebook: Decision_Tree_CART_vs_ID3.ipynb

## How to Run
1. Open the Colab link
2. Run all cells
3. Dataset is automatically loaded from GitHub
